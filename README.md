# Clawdbot Zeabur Deployment

Clawdbot remote AI assistant deployed on Zeabur.

## Environment Variables

Set these in Zeabur:

- `ANTHROPIC_API_KEY` - Your Anthropic API key
- `TELEGRAM_BOT_TOKEN` - (Optional) Telegram bot token
- `PORT` - Auto-set by Zeabur (default: 18789)

## Deployment

This project auto-deploys to Zeabur from this directory.

## Access

After deployment, access the gateway at:
`https://your-service.zeabur.app`

Get the tokenized URL via logs or dashboard.
